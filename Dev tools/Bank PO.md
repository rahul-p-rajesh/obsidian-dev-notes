### Data Analysis and Interpretation For Mains

|   |   |
|---|---|
|**Topic**|**Description**|
|**Simplification**|Solve expressions using BODMAS and basic arithmetic rules.|
|**Average**|Find the mean value of the given numbers or data sets.|
|**Percentage**|Calculate parts per hundred and compare values.|
|**Mixture and Allegations**|Solve problems involving mixing two or more entities with different ratios.|
|**Ratio and Proportion**|Compare quantities and solve scaling problems.|
|**Data Interpretation**|Analyse and solve questions based on charts, tables, or graphs.|
|**Mensuration and Geometry**|Calculate area, volume, perimeter, etc., of 2D and 3D shapes.|
|**Quadratic Equation**|Solve equations of the form ax² + bx + c = 0.|
|**Interest**|Calculate simple and compound interest.|
|**Problems of Ages**|Solve age-related word problems using algebra.|
|**Profit and Loss**|Compute selling price, cost price, and profit/loss percentage.|
|**Number Series**|Identify the missing number or the next number in a sequence.|
|**Speed, Distance, and Time**|Solve motion-related problems using formulas and unit conversion.|
|**Time and Work**|Calculate work efficiency, combined work, and time taken.|
|**Number System**|Solve problems based on divisibility, HCF, LCM, and basic number properties.|
|**Data Sufficiency**|Determine whether the given data is enough to answer a question.|
|**Linear Equation**|Solve a single or a pair of linear equations in one or two variables.|
|**Permutation and Combination**|Count possible arrangements or selections in different scenarios.|
|**Probability**|Calculate the chance or likelihood of a particular outcome.|

### Reasoning Syllabus for Mains

|   |   |
|---|---|
|**Topic**|**Description**|
|**Verbal Reasoning**|Questions that assess logical thinking and language-based reasoning.|
|**Syllogism**|Deduce conclusions based on given statements using logic rules.|

|**Code Inequalities**|Solve inequalities coded in symbols to find valid conclusions.|
