
### Terminal
1. terminal / terminal _emulator_ => A terminal emulator is a program that emulates a physical terminal. It's a program that lets you type commands into a window on your computer.
2. terminal is just a program that lets you issue text-based commands and renders the output of those commands.
### Shell
1. A shell is a program that takes commands provided by terminal, interprets them and runs them and returns the output if any
 2. Shells are often referred to as "REPL"s. REPL stands for
	- Read: Read the commands you type
	- Eval (evaluate): Evaluate those commands, usually by running _other_ programs on your computer
	- Print: Print the output of those commands
	- Loop: Give you a new prompt to type another command and repeat
3. There are different shell types: bash (windows), Zsh (macos)
